About CCTelemetry

CCTelemetry is a local, real-time desktop dashboard for Claude Code — token usage, cost, and per-tool/model/project/session breakdowns, living quietly in your menu bar. It reads the session logs Claude Code already writes on your machine; no API keys, no cloud services, nothing ever leaves your computer.
